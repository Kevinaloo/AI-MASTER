Kenya's first AI-powered trading platform — Crypto, Forex, Predictions & M-Pesa integrated.

## Setup Instructions

### 1. Upload to GitHub
- Go to your GitHub repo
- Upload all these files

### 2. Deploy to Netlify
- Go to netlify.com
- Click "Add new site" → "Import from Git"
- Choose your GitHub repo
- Add environment variables (see below)
- Click Deploy

### 3. Environment Variables (add in Netlify)
```
REACT_APP_SUPABASE_URL=https://xuhodkbusnprpwtkazmc.supabase.co
REACT_APP_SUPABASE_ANON_KEY=your_anon_key_here
```
